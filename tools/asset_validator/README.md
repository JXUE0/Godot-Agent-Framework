# Asset Validator

## Proposito
Validar assets segun estandares de nombres, tamanos, formatos y duplicados.

## Instrucciones
- Definir reglas por tipo (sprites, audio, UI, 3D).
- Reportar errores y warnings con rutas exactas.

## Ejemplos
- Rechazar archivos con espacios en el nombre.
- Alertar texturas mayores a 4096px.

## Limitaciones
- No modificar ni borrar assets automaticamente.
- Solo reportar y sugerir acciones.
