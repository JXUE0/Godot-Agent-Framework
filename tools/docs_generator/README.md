# Docs Generator

## Proposito
Generar documentacion tecnica automaticamente.

## Instrucciones
- Extraer metadata de escenas y scripts.
- Actualizar README de modulos.

## Ejemplos
- Generar listado de escenas y dependencias.
- Crear resumen de sistemas y autoloads.

## Limitaciones
- No reemplazar documentacion manual sin confirmacion.
