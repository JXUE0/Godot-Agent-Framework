# Scene Templates

## Proposito
Proveer plantillas de escenas Godot para acelerar creacion de UI, personajes, sistemas y niveles.

## Instrucciones
- Mantener templates en subcarpetas por categoria.
- Cada template debe incluir:
  - Scene (.tscn)
  - Script base (.gd)
  - README con uso

## Ejemplos
- `ui/menus/MainMenu.tscn`
- `characters/Player.tscn`

## Limitaciones
- No sobrescribir escenas existentes del juego.
- No crear escenas fuera de carpetas definidas por el proyecto.
