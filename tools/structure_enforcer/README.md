# Structure Enforcer

## Proposito
Verificar que el proyecto respete la arquitectura definida por el framework.

## Instrucciones
- Revisar ubicacion de escenas y scripts.
- Validar convenciones de nombres.

## Ejemplos
- Bloquear escenas UI fuera de `scenes/ui/`.
- Reportar scripts fuera de `scripts/`.

## Limitaciones
- No mover archivos automaticamente.
- Requiere aprobacion del Lead Engineer para cambios estructurales.
