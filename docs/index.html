Quickstart

Windows

.\scripts\setup_project.ps1 -ProjectRoot C:\ruta\tu_proyecto -InstallMCP -RunValidation
# Force MCP reinstall
.\scripts\setup_project.ps1 -ProjectRoot C:\ruta\tu_proyecto -InstallMCP -ForceMCP -RunValidation

macOS / Linux

./scripts/setup_project.sh /ruta/tu_proyecto --install-mcp --validate
# Force MCP reinstall
./scripts/setup_project.sh /ruta/tu_proyecto --install-mcp --force-mcp --validate

Then read docs/AI_GUIDE.md and follow the workflow.

MCP Integration

MCP runs a local WebSocket server inside Godot so AI tools can inspect scenes, scripts, and project settings safely. Install with the scripts above and enable the plugin in Godot.
