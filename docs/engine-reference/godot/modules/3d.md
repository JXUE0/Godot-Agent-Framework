# Godot 3D — Quick Reference
Last verified: 2026-02-12 | Engine: Godot 4.6

## Changes
- Jolt default for new projects (4.6).
- AgX tonemapper, SSR improvements.

## Patterns
- Use `WorldEnvironment` for post‑processing.
- Separate lights and probes into dedicated nodes.

## Common pitfalls
- Assuming Vulkan default on Windows.
- Using GodotPhysics‑specific properties without checking.
