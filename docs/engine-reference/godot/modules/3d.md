# Godot 3D — Quick Reference
Last verified: 2026-02-12 | Engine: Godot 4.6

## Cambios desde ~4.3
- Jolt es default 3D en proyectos nuevos (4.6).
- Mejoras SSR y tonemapper AgX (4.6).

## Patrones actuales
- Usar `WorldEnvironment` para post-proceso.
- Separar luces y probes en nodos dedicados.

## Errores comunes
- Asumir Vulkan default en Windows.
- Usar props especificas de GodotPhysics3D sin verificar motor activo.
