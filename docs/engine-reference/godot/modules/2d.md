# Godot 2D — Quick Reference
Last verified: 2026-02-12 | Engine: Godot 4.6

## Cambios desde ~4.3
- TileMapLayer reemplaza TileMap (4.3+).
- Rotacion de scene tiles en TileMapLayer (4.6).

## Patrones actuales
- Usar `Node2D.y_sort_enabled` en lugar de YSort.
- Separar colisiones en capas claras.

## Errores comunes
- Usar TileMap multi-layer en lugar de TileMapLayer.
- Usar YSort como nodo.
