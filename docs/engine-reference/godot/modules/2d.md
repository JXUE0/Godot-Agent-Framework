# Godot 2D — Quick Reference
Last verified: 2026-02-12 | Engine: Godot 4.6

## Changes
- TileMapLayer replaces TileMap (4.3+).
- Scene tile rotation in TileMapLayer (4.6).

## Patterns
- Use `Node2D.y_sort_enabled` instead of YSort.
- Separate collision layers clearly.

## Common pitfalls
- Using TileMap multi‑layer instead of TileMapLayer.
- Using YSort node.
