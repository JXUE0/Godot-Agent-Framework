# GODOT_BEST_PRACTICES.md

## GDScript
- Use static typing when possible.
- Name signals consistently.
- Scripts in `snake_case`.

## Scenes
- Separate UI, gameplay, and systems.
- Avoid huge scenes.

## Validation
- Run validators before delivery.

## Version reference
Before suggesting APIs, check `docs/engine-reference/godot/` (version, breaking changes, deprecated APIs).
