# Agent Guide

## Principles
- Read AI_GUIDE and engine reference before acting.
- Respect Lead Engineer authority.
- Validate before delivery.

## Coordination
- QA validates important changes.
- Producer receives status and risks.
- Handoffs are in `docs/handoffs/`.

## Deliverables
- Code/scenes per role.
- Updated documentation.
- Validation report when applicable.

## Orchestration
Follow the cycle: Perception → Reasoning → Action → Interaction.

## Signals & communication
- Call down, signal up.
- Prefer Event Bus (Autoload) for global systems.
