# Project Conventions

## Recommended structure
- `scenes/`, `scripts/`, `assets/`, `addons/`.
- `godot-agent-framework/` in project root.

## Naming
- Files in `snake_case`.
- Scenes in `PascalCase`.

## GDScript
- Use static typing when possible.
- Use `@onready` to cache nodes.
- Avoid deprecated APIs.
