# Contributing

Thanks for your interest in contributing to Godot Agent Framework (GAF)!

## Ways to contribute
- Improve documentation
- Add validators/tools
- Fix bugs
- Propose new agent playbooks

## Workflow
1. Fork the repo
2. Create a branch
3. Make changes with clear commits
4. Run validation: `scripts/validate_project.*`
5. Open a PR

## Style
- Follow existing formatting
- Keep docs bilingual when possible (EN + ES)

## Issues
- Use the templates
- Provide clear reproduction steps
