# Godot Agent Framework (GAF)

Framework drop-in para proyectos Godot 4.x que permite a cualquier IA trabajar de forma profesional con agentes, herramientas y estándares definidos.

## Instalación rápida (Drop-in)
1. Descarga o clona este repositorio dentro de tu proyecto Godot:
   `my_game/godot-agent-framework/`
2. Ejecuta el setup:
   - Windows: `scripts\setup_project.ps1`
   - macOS/Linux: `./scripts/setup_project.sh`
3. Lee `docs/AI_GUIDE.md` y sigue el flujo.

## Estructura
- `agents/` Roles y reglas de trabajo.
- `docs/` Guías oficiales del framework.
- `tools/` Herramientas internas (validadores, plantillas, generadores).
- `scripts/` Setup, validación y generación de docs.

## Objetivo
Que cualquier IA pueda integrarse a un proyecto Godot y ejecutar tareas con calidad profesional, respetando arquitectura, convenciones y colaboración entre roles.
